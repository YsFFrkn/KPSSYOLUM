Geliştirmiş olduğumuz proje C# programlama dilinde .Net Framework çatısı altında Visual Studio 2019 kullanılarak geliştirilmiştir. Verimi saklama tarafında ise popüler olan Microsoft SQL Server Management Studio uygulaması form uygulamamız ile paralel şekilde çalışmaktadır. Uygulamamız esnasında çeşitli tasarım prensip ve desenleri göz önünde bulundurularak geliştirmeler tasarlanmış ve gerçekleştirilmiş. 

Uygulamamız üç farklı kullanıcı çeşidine yönelik geliştirilmiştir, bunlar: Öğrenci, Öğretmen ve Admin.

Admin ve öğretmen kullanıcıları aynı form ekranları kullanmakta olup soru eklemek, mevcut soru havuzundaki soruları güncelleme yetilerine sahiptirler. Ayrıca Admin yetkisindeki kişiler eklenmiş soruları onaylama yetkisinede sahiptirler. Onaylanmamış sorular öğrencilerin karşılaşacağı sorular arasında bulunamaz.

Öğrenci kullanıcılarıda öncelikli olarak üye olabilip, soru havuzundaki soruları çeşitli parametreler içersinide çözme yetisine sahiptirler.

GitHub linkimizde yüklü olan .sln uzantılı Form dosyamızı ve .bak uzantılı MSSQL dosyamızı indirerek programımıza erişim sağalayabilirsiniz.
